# Workday Plus brand assets

Place your Home Assistant integration brand images in this folder.

## Required (minimum)

- `icon.png`
- `logo.png`

## Optional variants

- `dark_icon.png`
- `dark_logo.png`
- `icon@2x.png`
- `logo@2x.png`
- `dark_icon@2x.png`
- `dark_logo@2x.png`

## Recommended export settings

- PNG format
- Transparent background
- Square icon canvas for `icon*.png`
- Wider horizontal canvas for `logo*.png`
- Provide `@2x` at exactly 2x the base dimensions

## Typical practical sizes

- `icon.png`: 256x256
- `icon@2x.png`: 512x512
- `logo.png`: 512x256
- `logo@2x.png`: 1024x512

After adding images, restart Home Assistant and refresh browser cache to see the updated integration icon/logo.
